Customize the default wordpress login page, logo on top, form background color and login buttun style.

== Description ==

Login Page Tailor is an easy tool that provides you with several options to re-style the default wordpress login page.

== Installation ==

1. Upload `login-page-tailor.zip` to the `wp-content/plugins/` directory and unzip.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Check newly added page under settings navigation menu, "Login Page Tailor"
4. Head to the page to change available features for re-styling your login page.

== Frequently Asked Questions ==

= 1) Which style properties can I change with this plugin? =

At the moment you can change: "Login Logo" , "Logo sizes" , "Input form background color,border size,border color,border round corner radius" , "Buttun color, border color"

= 2) Can I reset setting back to default? =

At this stage the only way is deactivating the plugin, very soon there will be reset to default option availabe.
